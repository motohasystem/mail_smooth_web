# mail_smooth_web
https://motohasystem.github.io/mail_smooth_web/

## 概要
- メルマガを音声読み上げさせる際に快適に聞けるようにテキスト整形するためだけのツールです。
- 区切り線に使われる=============とか-------------とかの記号の繰り返しを除去します
- URL文字列を見つけると(URL)として短縮します。
- 指定した文字数でページを区切ります。


## 短縮対象となる記号の繰り返し

 ─────────────────
 ━━━━━━━━━━━━━━━━━
 ==================================
 ┏━━━━━━━━━━━━━━━━

などに今のところ対応しています。
